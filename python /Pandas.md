---
 ۷. پروژه کوچک

ساخت جدول نمرات و محاسبه معدل هر دانش‌آموز:

import pandas as pd

data = {
    "نام": ["علی", "سارا", "نیما"],
    "ریاضی": [18, 19, 15],
    "فیزیک": [17, 20, 14],
    "شیمی": [16, 18, 13]
}

df = pd.DataFrame(data)
df["معدل"] = df[["ریاضی", "فیزیک", "شیمی"]].mean(axis=1)
print(df)

📌 خروجی نمونه:

     نام  ریاضی  فیزیک  شیمی   معدل
0   علی    18     17     16   17.0
1  سارا    19     20     18   19.0
2  نیما    15     14     13   14.0
